基于非均匀散点拟合三维曲面

1.首先，构造数据

x = -2.5 + 5*rand([200 1]); % 范围[-2.5,2.5]
y = -2.5 + 5*rand([200 1]); % 范围[-2.5,2.5]
z = x.*exp(-x.^2-y.^2);

2.定义一个规则网格，并计算每个网格点的取值\((xq,yq,zq)\)

[xq,yq] = meshgrid(-2:0.2:2, -2:0.2:2);
zq = griddata(x,y,z,xq,yq,'cubic');

3.绘图

hold on
plot3(x,y,z,'.')
mesh(xq,yq,zq)
view(3)
hold off
